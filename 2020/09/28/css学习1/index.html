Padding

It adds/removes space inside of the element but around the content box.

Unable to accept negative values

Margin

It adds/removes space around the element

Able to accept negative values

Margin: Auto (居中对齐)

First you have to define the width, then set the margin

eg:

width: 500px;
margin: 0 auto
