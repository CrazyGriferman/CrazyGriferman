Terminus和Origami的作用

Terminus插件用于在sublimetext中通过调用系统的终端来运行程序。只不过它不会弹出终端那个程序，而是在sublimetext内部调用，这样就解决了python的指令交互问题。

Origami这个插件用于创建新的窗口。它可以设置各种窗口弹出的位置。

配置流程

通过package control中的Install Package来安装Terminus和Origami这两个插件。

安装好了之后，我们需要找到sublime text自带的python的运行系统。

打开指令框，输入view，选择view package file，

之后输入 sublime-build python
