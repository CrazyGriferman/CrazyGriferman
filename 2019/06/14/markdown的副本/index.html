Markdown的一些高频使用的语法

加粗

方法：

在段落前后各添加两个“*”或者两个“_”

示例：

**ABC** _ABC_

效果：

ABC

快捷键：

选中段落+command+b

倾斜

方法：

在段落前后各添加一个“*”或者一个“_”

示例：

*ABC* _ABC_

效果：

ABC

引用

方法：在段落的开头添加一个“<”

示例：

> 这是用来测试的句子

效果：

> 这是用来测试的句子

添加行内代码

方法：找到键盘左上角的“～”健，在字母输入法下按两下，在其中添加内容，如”`ABC`“

示例:

`ABC`

效果:

ABC

添加标题

方法：在文字前面添加”#”，总共有六级标题,”#”添加得越多，级数越大，标题大小越小

示例：
#测试 一级标题
##测试 二级标题
###测试 三级标题
#####测试 四级标题
#####测试 五级标题
######测试 六级标题

效果：

测试 一级标题

测试 二级标题

测试 三级标题

测试 四级标题
测试 五级标题
测试 六级标题

在一级和二级标题下，系统会自动创建两根很细的分割线，而三、四、五、六级标题则没有

符号”"的功能

方法：如果我们在markdown文章中想要使用* - 等符号却发现会被转化，我们只需在想用的符号前加一个”\“符号
示例：
\> 测试句子

效果：> 测试句子

添加分割线

方法：新起一行并在这行中至少打三个-

示例：

- - -

效果：

添加子父集列表

方法：
示例：

* 测试
　　+ 测试
　　　　- 测试

效果：

  • 测试
    • 测试
      • 测试

注： *、+、- 这三个符号都可以，但是在符号和文字间需要有个空格才能实现。
注：建立子集时，需要在符号前按两下tab（如“+ 测试”）,而二级子集需要按四下tab（如“- 测试”）

在Mweb中创建超链接时可能存在的问题

给文字添加超链接

在笔者写这篇博客时，发现在给文字添加超链接时有一些问题

一般来说，“超链接”的功能可以由以下方式实现：

1.[你的文字](你的url)

2.<你的url>

但根据mweb官网中的文档

笔者在撰写这篇文章时发现，若使用2的办法，你的url之前必须要有“http://”而不能是简写的网址。除此之外，笔者还发现其实可以不用添加<>,直接打一个网址就行，例如你可以直接打 www.baidu.com ，此时也不用加任何额外的符号，但是请注意这样打出来的网址的前后必须要有空格，若是网址开头前没有空格，系统默认此不是一个网址，具体效果如下：

测试www.baidu.com

注: 虽然在Mweb中该网址并不显示为超链接，但当文章发布到博客上时网站还是显示为超链接的。

而如果是后面没有，则会产生如下效果：

www.baidu.com后面的字就会和网址连起来并且被识别为这个网站地址的一部分。

这样的话网站也就打不开了。

因而对于mweb，我们可以有三种方式创建超链接：

1.[你的文字](你的url)

这里的url可以不加“http://”

2.<你的url>

此处的url之前必须要有”http://“

3.直接打出url
例: www.baidu.com

此处网站地址的前后必须要有空格（如果首尾都有文字连接的话）

关于插入图片我想多说几句

如果我们想让插入在markdown中的图片在网页中呈现出来，图片所用的地址就必须是网络地址，而不能是本地地址，这也就是为什么很多人会将图片上传至图床的原因，这样算作“在网上保存了图片”并创建了可以呈现这张图片的网址。

对于轻量级用户来说，如果你上传的图片容量比较小且不是很多的话，强烈推荐你以下方案：

图床: SM.MS （免费无需注册）
网址:

https://sm.ms

这个网址支持小于5mb的图片，每次上传最多十张
你可以直接把图片上传到该网站并获得url，但这样效率太低，也就是为什么我们需要图片上传的软件

图片上传软件: PicGo （免费无需注册）
下载地址:

https://github.com/Molunerfinn/PicGo/releases

别忘记网址的首尾空格

下载完软件后打开
你会发现在mac的右上角有这么一个小图标

右键并打开详细窗口

在图床设置这一栏中我们选择设置SM.MS为默认图床

建议1:在picgo设置中修改快捷键

建议2:在插件设置中搜索autocopy并下载，该插件可以在你上传完一张图片后自动将网址复制在你的剪贴板上，你可以直接按command+v进行粘贴

如何上传图片并获取图片的url？

Picgo有两种上传方法。

方法一:

1.将文件拖拽到上传区中上传

2.在上传成功后，在相册中找到那张之前上传的照片

3.点击该照片底下从左往右数第一个按钮

该按钮是用来复制图片的url

从左往右数第二个是用来修改这张图片的url，第三个是从相册中移除这张图片

即使你移除，在图床上它也不会消失，只是在相册中不显示了，url还是在的

在做完这三步之后，你就可以在mweb中将图片地址粘贴了，该图片即便发布到网上也会显示

方法二:
1.复制你想上传的照片 ，即选中图片后按 command+c

2.再按你之前设置的上传快捷键

3.如果你之前安装了autocopy的插件，那么你可以直接command+v将图片地址复制到mweb中

如果没有安装，你需要打开相册找到那张图片并点击从左到右第一个建复制它的url

相比之前还是方法二简单，所以请熟记一个快捷键并安装个插件吧！这样会很大地提高图片上传的效率。
